ScrollReveal adalah pustaka JavaScript untuk menganimasikan elemen dengan mudah saat masuk/keluar dari viewport. Itu dirancang agar kuat dan fleksibel, tetapi mudah-mudahan Anda akan terkejut di bawah betapa mudahnya mengambilnya.

popmotion

Kotak alat JavaScript animator. PowerfulMendukung keyframe, pegas dan animasi inersia pada angka, warna, dan string kompleks. Tingkat rendahSederhana, fungsi yang dapat dikomposisi, portabel untuk lingkungan JS apa pun. Pustaka di belakang pustaka, itu menggerakkan animasi di Framer Motion. StableWritten dalam TypeScript dan menikmati lebih dari 95% cakupan pengujian. TinyThe animate function kurang dari 5kb, dan setiap fungsi utilitas dapat diimpor secara individual.

Lottie JS

Lottie untuk Android, iOS, Web, React Native, dan Windows Lottie adalah perpustakaan untuk Android, iOS, Web, dan Windows yang mem-parsing animasi Adobe After Effects yang diekspor sebagai json dengan Bodymovin dan merendernya secara asli di seluler dan di web! Untuk pertama kalinya, desainer dapat membuat dan mengirimkan animasi yang indah tanpa seorang insinyur dengan susah payah membuatnya ulang dengan tangan. Mereka mengatakan sebuah gambar bernilai 1.000 kata, jadi ini dia Animasi di atas dibuat di After Effects, dan dapat dirender secara native di semua platform dengan file json sederhana.

mojs

perpustakaan grafik gerak javascript yang cepat, siap untuk retina, modular, dan open source. Dibandingkan dengan perpustakaan lain, ia memiliki pendekatan struktur animasi kode dan sintaks yang berbeda. API deklaratif memberi Anda kontrol penuh atas animasi, membuatnya dapat disesuaikan dengan mudah. Pustaka menyediakan komponen bawaan untuk memulai animasi dari awal seperti html, bentuk, pusaran, burst, dan terhuyung-huyung, tetapi juga memberi Anda alat untuk membantu membuat animasi dengan cara yang paling alami. Menggunakan mojs di situs Anda akan meningkatkan pengalaman pengguna, memperkaya konten Anda secara visual, dan membuat animasi yang menyenangkan dengan tepat
